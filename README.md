Jared Chan's personal portfolio, a single self-contained page at [www.jaredchan.ai](https://www.jaredchan.ai). Everything on the site is matter from the particle sphere.

- `PORTFOLIO-EFFECTS.md` is the standing brief for the visual effects. Read it before changing anything.
- Hosted on Firebase Hosting, deployed with `firebase deploy`.
